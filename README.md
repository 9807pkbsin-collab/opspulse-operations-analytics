# OpsPulse – Enterprise Operations Analytics Platform

## Overview

OpsPulse is a data analytics project designed to monitor operational KPIs and provide real-time insights through automated data pipelines and dashboards.

## Objective

The goal of this project is to centralize operational data from multiple sources and enable real-time KPI tracking and performance analysis.

## Tools & Technologies

* Python (Pandas, NumPy)
* SQL (MySQL)
* Power BI
* ETL pipelines
* Data visualization

## Project Workflow

1. Extract operational data from databases
2. Transform and clean the data using Python
3. Load data into analytical tables
4. Build KPI dashboards in Power BI

## Key Insights

* Identified operational bottlenecks affecting performance
* Automated reporting workflows
* Reduced manual reporting time significantly
